# Armin & Christina Wedding Guest Site

A mobile-first one-page guest website for the August 5, 2026 wedding at Secrets Moxche Playa del Carmen.

## Included

- Hero photo and personalized welcome
- Four quick links: schedule, ceremony location, photo upload, and resort map
- Wedding schedule open by default
- Ceremony directions and unplugged-ceremony note
- Dress code and practical tips
- Shared Google Drive photo folder
- External full-size resort map link
- Dining, transportation, amenities, and essential information
- Downloadable calendar event (`armin-christina-wedding.ics`)
- Mobile bottom dock for schedule and photo uploads

## Files

- `index.html` - main website
- `armin-christina-wedding.ics` - downloadable calendar event
- `images/hero-resort.jpg` - hero background image
- `images/resort-map.jpg` - retained source image, not displayed because its labels should not be relied on for guest navigation
- `vercel.json` - Vercel routing configuration

## Before publishing

1. Test the Google Drive folder in an incognito/private browser and confirm guests can upload files.
2. Confirm the wedding schedule and ceremony directions with the wedding planner.
3. Test the downloadable calendar event on iPhone and Android.
4. Test the site outdoors on a phone at full brightness.
5. Redeploy the folder to the existing Vercel project.
