# Armin & Christina Wedding Guest Site

## Updated version

This package includes the redesigned Liquid Glass version of the guest website.

### Main changes
- Neutral black, pearl, ice-blue, and champagne Liquid Glass design
- Hero image repaired with a root-level image and an `images/` fallback
- Transportation section removed
- Add-to-calendar menu with Apple/Outlook and Google Calendar options
- Updated full-property resort map displayed directly on the page
- Preferred Club and Impression-only restaurant access clearly labeled
- Google Drive photo-upload link preserved
